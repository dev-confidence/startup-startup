StartUp-StartUp

The aim of this project is to StartUp a StartUp with a working example Continuous Integration & Delivery pipeline.

Latest Technologies & Services used:

  • Github
  • Travis CI

Continuous Integration & Deploymeny

  1. Write Tests & Code (TDD)
  2. Commit & Push to Github
  3. Travis CI builds & runs all tests (unit, functional, acceptance)
  4. If step 3 is successful: a. Create Git Tag / Release on Github (these are release candidates) b. Deploy Git Tag to CI environment
  5. Test on CI environment
  6. Deploy to Test environment from any tag / release as required
